# Модель OSI: 7 уровней с подробным описанием

Модель OSI (Open Systems Interconnection) — это концептуальная модель, которая описывает, как данные передаются в сети поэтапно через семь уровней.
Каждый уровень отвечает за свои функции и взаимодействует только с соседними уровнями.
