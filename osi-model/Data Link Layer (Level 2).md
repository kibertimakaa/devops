## 2. Канальный уровень (Data Link Layer)

### Основные функции:
- Обеспечивает надежность передачи данных.
- Формирует фреймы (блоки данных) из битов.
- Проверяет целостность полученных данных, исправляет ошибки.
- Управляет доступом к среде передачи (например, к кабелю).
### Примеры:
- Ethernet (MAC-адрес).
- Wi-Fi, PPP (Point-to-Point Protocol).
### Детали:
- Включает подуровни LLC (Logical Link Control) и MAC (Media Access Control).
